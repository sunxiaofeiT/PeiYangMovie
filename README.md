# Pei Yang Movie

北洋电影社团管理系统前端(Ember version)

## Run 

1. `git clone <repo>`
2. `cd <project>`
3. `npm install`
4. `npm install -g ember-cli`
5. `ember serve`
6. Visit [localhost:4200](http://localhost:4200)

## Build

* `ember build` (development)
* `ember build --environment production` (production)
